# 比赛数据集说明文档

## 一、数据集概述

本数据集用于本次比赛，仅供参赛团队在比赛期间进行模型训练、验证与测试使用。数据集包含 **25 个类别**，涵盖舰船、飞机及发射车目标，标注格式为**水平边界框**。

## 三、标注格式

标注文件为YOLO格式的**水平边界框**，每个`.txt`文件与对应图像同名，每行格式如下：
<类别ID> x_center y_center width height

- 坐标均为归一化后的浮点数（0~1）
- `x_center, y_center` 为目标框中心点坐标
- `width, height` 为目标框的宽和高
